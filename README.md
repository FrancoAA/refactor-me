# REFACTOR-ME

The objective of this exercise is see how well you know Angularjs. Your task consist of refactor this Angularjs app using the best practices, reduce the code duplication, fix the bugs and write unit-tests.

- in order to execute run a project it has to be configured
- only adminstrators can configure a project
- a basic user cannot run a project
- only the owner of the project can delete it
